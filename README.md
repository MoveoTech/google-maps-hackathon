# DayTrip App

DayTrip is a mobile App for planning a day trip using Google Maps APIs


## Run project in development

Using [React-Native](https://reactnative.dev/) and [Expo.io](https://expo.dev/)

```bash
npm install
```

```bash
npx expo start -c
```
